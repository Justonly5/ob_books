
## Deployment&DaemonSet

| **维度** | **Deployment** | **DaemonSet** |
| ------ | -------------- | ------------- |
| Pod 数量 | 固定副本数（可扩缩）     | 每个 Node 一个    |
| 调度方式   | Scheduler 随机调度 | 每个 Node 强制一个  |
| 流量入口   | 集中入口           | 分布式入口         |
| 网络路径   | 可能跨节点          | 通常本地处理        |
| 典型场景   | 云 LB / 小规模     | 高性能 / 边缘入口    |
